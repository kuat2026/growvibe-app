# PRD: экран настроек напоминаний об уборке

## Контекст

Часть мобильного приложения для уборки квартиры (чек-лист по комнатам + профиль с прогрессом). Этот экран отвечает за настройку push-напоминаний: когда и как часто пользователь получает сигнал о том, что пора убираться.

## Цель

Дать пользователю возможность настроить расписание напоминаний об уборке так, чтобы он не забывал убираться, но и не уставал от уведомлений.

## Пользовательская история

Как пользователь приложения, я хочу настроить дни недели, время и условия повторных напоминаний, чтобы получать уведомления об уборке в удобном для меня ритме.

## Структура экрана

### Шапка
- Кнопка "назад" (стрелка влево)
- Заголовок "Напоминания"

### Блок 1 — основной переключатель
- Тумблер "Напоминать об уборке"
- Подпись: "Push-уведомление по расписанию"
- При отключении остальные настройки экрана становятся неактивными (не входит в текущий макет, уточнить при разработке)

### Блок 2 — дни недели
- Заголовок "Дни недели"
- 7 круглых кнопок-переключателей: Пн, Вт, Ср, Чт, Пт, Сб, Вс
- Множественный выбор (можно отметить несколько дней)
- Выбранный день — заполненный кружок (акцентный цвет), невыбранный — кружок с тонкой обводкой
- В макете по умолчанию выбрана Сб

### Блок 3 — время
- Заголовок "Время"
- Поле-кнопка с иконкой часов и текущим значением (например, "11:00")
- Нажатие открывает выбор времени (тайм-пикер, отдельный экран/модалка — не в текущем макете)

### Блок 4 — повторные напоминания
- Заголовок "Если уборка пропущена"
- Тумблер "Повторное напоминание через 3 дня"
- Тумблер "Звук уведомления"

### Блок 5 — напоминания по комнатам
- Заголовок "Напоминать по комнатам отдельно"
- Тумблер "Ванная и туалет чаще (раз в 3 дня)" — отдельное расписание для зон с более частой уборкой

### Нижняя часть
- Кнопка "Сохранить" (на всю ширину, акцентный цвет фона)

## Функциональные требования

1. Пользователь может выбрать один или несколько дней недели для основного напоминания
2. Пользователь может задать время отправки напоминания
3. Пользователь может включить/выключить повторное напоминание при пропуске уборки и задать его периодичность (в макете — 3 дня, фиксированное значение для первой версии)
4. Пользователь может включить/выключить звук уведомления
5. Пользователь может включить отдельное, более частое напоминание для влажных зон (ванная, туалет) — фиксированный интервал 3 дня для первой версии
6. Все изменения сохраняются по нажатию кнопки "Сохранить"
7. Главный тумблер "Напоминать об уборке" — мастер-переключатель для всего экрана

## Нефункциональные требования

- Внешний вид соответствует общему стилю приложения (минимализм, без градиентов, плоские поверхности)
- Поддержка светлой и тёмной темы
- Экран адаптирован под мобильный формфактор (мокап — iOS-стиль)

## Вне рамок текущей версии (требует отдельного прода)

- Поведение экрана при выключенном мастер-тумблере (блокировка/скрытие остальных настроек)
- Сам тайм-пикер (компонент выбора времени)
- Настраиваемая периодичность повторного напоминания и напоминания по влажным зонам (сейчас фиксированные значения)
- Логика самих push-уведомлений (бэкенд, разрешения ОС)

## Связанные экраны

- Главный экран приложения (список комнат, общий прогресс)
- Профиль/чек-лист по комнатам с сохранением прогресса между сессиями
